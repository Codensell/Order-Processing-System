# Order Processing System (ASP.NET Core + PostgreSQL + Kafka + Docker)

## Описание

Данный проект представляет собой упрощённую систему обработки заказов, реализованную в виде Web API.

Цель проекта — продемонстрировать базовые навыки backend-разработки:

* работа с REST API
* взаимодействие с базой данных (PostgreSQL)
* использование продюсера сообщений (Kafka)
* контейнеризация приложения (Docker)

Проект разрабатывается поэтапно с постепенным усложнением архитектуры.

## Архитектура

Проект построен с разделением на слои:

* **API** — обработка HTTP-запросов
* **Application** — бизнес-логика
* **Domain** — модели предметной области
* **Infrastructure** — работа с БД и внешними сервисами

Такой подход позволяет:

* уменьшить связанность компонентов
* упростить тестирование
* обеспечить масштабируемость

## Используемые технологии

* ASP.NET Core Web API
* PostgreSQL
* EF Core
* Apache Kafka
* Docker / Docker Compose

## Функциональность

### Этап 1 — Базовый API + БД

* Создание заказа
* Получение заказа по ID
* Получение списка заказов
* Сохранение данных в PostgreSQL

### Этап 2 — Интеграция с Kafka

* Публикация события при создании заказа
* Обработка событий через консюмер

### Этап 3 — Docker

* Контейнеризация приложения
* Поднятие окружения через docker-compose:

  * API
  * PostgreSQL
  * Kafka

### Этап 4 — Расширение логики

* Добавление статусов заказа (Created, Paid, Cancelled)
* Имитация оплаты
* Генерация дополнительных событий

## Примеры API

### Создание заказа

POST /orders

### Получение заказа

GET /orders/{id}

### Получение списка заказов

GET /orders


## Запуск проекта

```bash
docker-compose up --build
```

После запуска API будет доступен по адресу:

```
http://localhost:5000
```
